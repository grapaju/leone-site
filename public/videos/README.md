# Vídeos do Site

## Hero Home Page
Adicione o vídeo principal da home page aqui com o nome:
- `hero-home.mp4`

**Recomendações:**
- Formato: MP4 (H.264)
- Resolução: 1920x1080 (Full HD)
- Duração: 10-30 segundos (loop perfeito)
- Tamanho: < 10MB (otimizado)
- Conteúdo: Vista aérea de Balneário Camboriú, fachadas de empreendimentos Leone Pavan, ou detalhes arquitetônicos

**Enquanto não adicionar o vídeo:**
O player tentará carregar mas não mostrará erro. O overlay escuro garantirá que o texto permaneça legível.
